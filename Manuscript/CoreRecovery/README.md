# ITSM Core Recovery manuscript

This directory contains the modular v12 research-stage reconstruction. It does
not replace `Manuscript/ITSM_Core_Cosmology_v11.4.1.tex`.

Main file:

```text
ITSM_Core_v12.0-alpha.1.tex
```

Build from the repository root with the bundled Codex LaTeX compiler, or from
this directory with an available `pdflatex`/`latexmk` installation.

The manuscript intentionally distinguishes derived, conditional, open and
rejected claims. `13/12`, `H0=72.97`, automatic Solar-System compliance and the
historical SPARC `p=0.62` are not live predictions in this draft.
