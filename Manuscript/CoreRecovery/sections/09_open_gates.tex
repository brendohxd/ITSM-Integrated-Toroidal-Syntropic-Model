The dependency order is
\begin{equation}
 \mathrm{UVIR}\to\mathrm{MAT}\to(\mathrm{SCR},\mathrm{LEN})
 \to\mathrm{DISK}
 \to(\mathrm{TOP},\mathrm{VOR},\mathrm{WAK},R)
 \to\mathrm{CBR\mbox{-}002}
 \to(\mathrm{COS},\mathrm{PERT}).
\end{equation}

This ordering is deliberately restrictive.  For example, running CBR-002 with
an arbitrary source adjusted to hold $13/12$ would test a tuning procedure,
not ITSM.  Likewise, fitting the algebraic disk law more aggressively would
not answer whether the actual periodic field equation changes the inferred
coupling.
