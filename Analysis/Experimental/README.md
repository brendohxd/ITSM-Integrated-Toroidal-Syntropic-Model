# ITSM — Experimental Scripts Staging Area

All new scripts are developed here first.
Nothing in this folder is referenced by the manuscript or committed as primary results.

## Protocol
- New script developed here ? tested ? peer-reviewed feasibility assessed
- Only promoted to Scripts/ after explicit author approval
- Each sub-experiment gets its own subdirectory
